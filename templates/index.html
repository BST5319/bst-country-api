This is a basic API used to fetch country capital using the name of country

http://bst-country-api.onrender.com/India/{State_name}
http://bst-country-api.onrender.com/India/Maharashtra

Note: While using state names having two words use the state names as ArunachalPradesh , AndhraPradesh , etc...

Example: http://bst-country-api.onrender.com/India/ArunachalPradesh
